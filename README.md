# HalfSnap - 6寸相纸对半切工具

为小米照片打印机1S量身定制的在线工具，助您轻松将两张照片合并为一张6寸相纸模板，打印后沿中间裁剪线即可获得两张标准4:3比例照片。

## 功能特点

### 📸 智能照片处理
- **双图上传**：支持同时上传两张照片
- **自动旋转**：竖版照片自动旋转为最佳显示方向
- **手动调整**：支持90°旋转和重新选择图片
- **拖拽上传**：支持拖拽方式上传图片

### 🖨️ 打印优化
- **完美比例**：输出比例25:37，适配6寸相纸100mm×148mm规格
- **细边框设计**：每张照片采用4:3比例显示，带有精致边框
- **裁剪指示**：中间虚线裁剪线，打印后沿虚线裁剪即可

### 📐 多分辨率输出
| 分辨率 | 尺寸 | 适用场景 |
|--------|------|----------|
| 2K高清 | 1440×2160 | 手机屏幕查看 |
| 4K超清 | 2880×4320 | 高质量打印 |
| 打印优化 | 1800×2700 | 打印机专用 |

### 💻 跨设备支持
- 完美适配桌面端和移动端
- 响应式布局，界面美观简洁
- 无需安装任何软件，浏览器即用

## 使用方法

### 1. 上传照片
- 点击上传区域或拖拽图片到对应位置
- 支持 JPG、PNG 等常见图片格式
- 建议上传3:4比例的竖版照片效果最佳

### 2. 调整照片
- 如需旋转照片，点击旋转按钮（🔄）
- 如需更换照片，点击重新选择按钮（📂）
- 系统会自动识别竖版照片并旋转

### 3. 选择分辨率
- **2K高清**：适合日常查看，文件体积适中
- **4K超清**：适合高质量打印输出
- **打印优化**：专为打印场景优化

### 4. 生成并下载
- 点击"生成相纸模板"按钮
- 预览合并效果
- 点击"下载相纸模板"保存图片

### 5. 打印裁剪
- 将图片发送至小米照片打印机1S打印
- 打印完成后，沿中间虚线裁剪
- 获得两张标准4:3比例照片
